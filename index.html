Projetos

Machine Learning
Este projeto demonstra um pipeline completo de machine learning focado em classificação multiclasse utilizando o modelo Gaussian Naive Bayes (GNB). O principal objetivo é explicar o desempenho do modelo por meio de técnicas de interpretabilidade (XAI). Para treinar e otimizar o modelo, foram utilizados diversos métodos de ajuste de hiperparâmetros.
• Python • Evaluation metrics
Smart Routine
Com o crescimento do home office e o excesso de notificações digitais, muitas pessoas têm dificuldade em manter uma rotina equilibrada, produtiva e saudável. SmartRoutine AI é um assistente pessoal inteligente desenvolvido em Python e possui integração com e-mail e Google Calendar, capaz de compreender linguagem natural, analisar humor, organizar tarefas e ensinar rotinas saudáveis.
• Python • NLP
FixTime
FixTime é um projeto de gestão para oficinas, realizado em uma disciplina de Engenharia de Software. O projeto é uma plataforma eficiente que facilita o gerenciamento de horários para oficinas e torna mais fácil para os clientes encontrarem prestadores de serviços. Com um sistema intuitivo e responsivo, otimiza o agendamento e melhora a experiência para ambas as partes.
• PHP • Full Stack
Aquário Automático
O projeto consiste no desenvolvimento de um aquário automatizado, integrando o microcontrolador ESP32 para controlar a alimentação dos peixes, monitorar a temperatura da água e regular a iluminação de forma inteligente. O sistema também envia dados em tempo real para o usuário, permitindo o monitoramento remoto das condições do aquário por meio de um aplicativo ou plataforma web.
• C++ • Full Stack
